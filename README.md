## Quick Start

1. Install this extension from VS Code Marketplace
2. Install SUB compiler: [Download SUB](https://github.com/subhobhai943/sub-lang)
3. Create a new file with `.sb` extension
4. Start coding!

## Example

```sub
# Variable declarations with blockchain-style #
#var name = "Alice"
#var age = 25

# Function definition
#function greet()
    #print("Hello from SUB!")
#end

# Call the function
greet()
```

## Commands

- **SUB: Compile Current File** - `Ctrl+Shift+B` (Windows/Linux) or `Cmd+Shift+B` (Mac)
- **SUB: Run Current File** - `Ctrl+Shift+R` (Windows/Linux) or `Cmd+Shift+R` (Mac)

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for "SUB Language"
4. Click Install
